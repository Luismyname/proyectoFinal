Bienvenidos a mi proyecto personal de fin de curso.

La finalidad de está aplicación es facilitar el proceso de almacenaje y manejo de datos de nuestros clientes y empleados de una clinica en psioterapia.

Es un proyecto el cual se estructura de la mejor manera posible para que sea facilmente adaptable a otros entornos de manejo de clientes, empleados y facturacion,
ademas de una agenda para poder programar citas con antelacion y asi con un recordatorio poder cerrar la mayor cantidad de citas posibles.

Es un proyecto Ambicioso el cual intento estar desarrollando contantemente sus mejoras.

Este proyecto se realiza con las sigientes tecnologias:
1.- HTML
2.- CSS
3.- JAVASCRIPT
4.- JSON
5.- NODE.js


ATENCION!!!
ya que es un producto el cual esta en su face de construccion es importante destacar que para su prueba es necesario 
1.- Realizar un (npm start) en el servidor para asi poder inicializar la base de datos 
2.- Realizar un (npm start) en el area de cliente ya que es una aplicacion de escritorio manejada con electron.

Donde nos encontramos?

Por ahora en el proyecto llevamos las siguientes funciones implemetadas:
1.- Una interfaz de inicio de seccion para el usuario con usuario y contraseña, este iniciara seccion en un diferente html dependiendo del rol del usuario y estara limitado para cada ocasion.
2.- En el apartado de administrador obtenemos un primer vistazo de el manejo que podremos tener dentro de la aplicacion, siendo este un vistazo rapido de empleado, clientes, calendario y facturacion y otro menu lateral el cual podremos entrar en mas profundidad
